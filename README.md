# ENVS-193DS_winter-2026_final
Final Assignment for ENVS 193 DS Class @ UCSB

## General information

**What is this repo for?**  
This repo contains files demonstrating Pearson and Spearman's correlation test, personal data visualizations, affective visualizations, and a statistical critique of the paper of choice. 

**Whose repo is it? Who is contributing to it?**  
This is a repo I've created. I am the sole contributor.

## Data and file overview

**What files are there?**  
All code files are in the `code` folder. The code for the website is in the `homework_3.qmd` file.

All data files are in the `data` folder. The salinity and pickleweed biomass data are in the `salinity-pickleweed.csv` file, and the personal data for screen time and multiple predicting factors (day type, etc.) is in the `env193ds_personaldata.csv` file.  

**What code is there, and what does it do?**  
The code in the `final.qmd` file demonstrates how to check assumptions before Pearson/Spearman's correlation test, use a correlation test to examine the strength of the relationship between two variables, and data visualization of personal data on screen time.
